¿Que es un recurso Frontend y cual es su funcion?



Los recursos Frontend son aquellos que conforman la estructura visual de la pagina web, es decir, lo que el usuario ve, la posicion de los objetos y los estilos utilzados en la pagina web, tambien permite que el usuario interactue con la pagina con elementos como botones, imagenes y metodos de navegacion, todo esto permite tener una experiencia fluida y de calidad para el usuario.







¿Cuáles son las principales tecnologías utilizadas en el Frontend y cual es su importancia?



1. HTML

HTML es el núcleo del desarrollo frontend. Gracias a esta tecnología, se puede crear la estructura de un sitio web. HTML permite implementar imagenes botones, formularios, etc.

2. CSS

CSS es una tecnología que crea estilos adicionales a la estructura HTML. Ademas permite que una aplicación web sea responsiva (adaptable al tamaño del dispositivo) y crear interacciones gracias a animaciones.

3. JavaScript

JavaScript es un lenguaje de programación que te permite implementar funciones complejas en páginas web. De modo que cualquier característica que este del lado del frontend puede utilizar JavaScript. Ya sean actualizaciones de contenido, mapas interactivos, animación de Gráficos 2D/3D, etc., JavaScript está involucrado.
