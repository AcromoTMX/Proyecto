¿Que es la responsividad?




El sitio web responsivo está desarrollado para adaptarse a los equipos móviles, como Smartphones y tablets. Es recomendable tener una pagina web que tambien se adapta a dispositivos moviles ya que esto permite tener un mayor margen de publico y usuarios. Estos son algunos de los mayores beneficios:

1- El contenido se adapta al tamaño de la pantalla.

2- Los elementos se ajustan automáticamente para un desplazamiento vertical.

3- El menú del sitio también se adapta al dispositivo usado.

4- Es posible seleccionar los contenidos destacados para el móvil.

5- El tamaño de las imágenes también se adapta al diseño





¿Que es y para que sirve la accesibilidad web?




La accesibilidad web es la característica de los productos digitales que permite a los usuarios con alguna discapacidad navegar por sus interfaces con la menor dificultad posible. Esto puede traer los siguientes beneficios:

1. Promueve la inclusividad digital: El grupo menor de personas con discapacidades tienen una experiencia mas dificil y peor debido a las limitaciones de estos, las webs accesibles facilita el uso de estas por parte de este grupo.

2. Amplía el alcance de una marca: debido a que la mayoría de las experiencias digitales se pensaron para personas sin discapacidades, promover la accesibilidad es una nueva oportunidad de que las marcas lleguen a nuevos públicos que fueron descuidados por sus competidores.

3. Mejora el posicionamiento en buscadores: esto como producto de la aspiración por hacer del internet un lugar de acceso universal. Si el sitio es accesible para todos, habra un beneficio notorio en el SERP (Search Engine Results Page).





¿Como aplico la accesibilidad web?




Estas son algunas formas de implementar la accesibilidad web:

1. Usa alternativas de texto.

2. Buscar que el contenido sea multimedia.

3. Crear contenido fácil de ver y escuchar.

4. Asegúrarse de que el sitio se pueda navegar con medios alternativos.

5. Permitir la personalización de la experiencia.
